# 美容院运营 Agent 系统

这是一个专为美容院设计的智能运营助手系统，帮助美容院进行线上线下的宣传和运营管理。

## 📁 项目结构

```
.
├── brand_tone.md                    # 品牌调性指南
├── steering/                        # Agent 指导文档目录
│   ├── beauty-salon-operations.md   # 运营总指南（自动加载）
│   ├── content-templates.md         # 内容模板库
│   ├── marketing-calendar.md        # 营销日历
│   └── customer-service-guide.md    # 客户服务指南
└── README.md                        # 项目说明文档
```

## 🎯 Agent 核心功能

### 线上运营
- **社交媒体内容创作**：小红书、抖音、微信公众号、朋友圈
- **私域流量运营**：社群管理、会员体系、客户维护
- **线上推广活动**：新客活动、转介绍、节日营销

### 线下运营
- **门店氛围营造**：装饰、音乐、动线优化
- **客户体验提升**：接待流程、服务话术、档案管理
- **线下活动策划**：沙龙讲座、VIP答谢会、体验会

## 🚀 快速开始

### 1. 内容创作
向 Agent 提问示例：
- "帮我写一篇小红书文案，介绍补水项目"
- "给我设计一个抖音视频脚本，展示面部护理过程"
- "写一篇微信公众号文章，讲解春季护肤知识"

### 2. 活动策划
向 Agent 提问示例：
- "帮我策划一个三八妇女节的营销活动"
- "设计一个新客体验活动方案"
- "制定本月的朋友圈发布计划"

### 3. 客户服务
向 Agent 提问示例：
- "客户问价格太贵，我该怎么回复？"
- "如何处理客户说没效果的投诉？"
- "给我一个预约确认的话术"

### 4. 运营分析
向 Agent 提问示例：
- "分析一下我们的客户流失原因"
- "给我一个会员分级管理方案"
- "如何提高客户复购率？"

## 📚 文档说明

### brand_tone.md
定义美容院的品牌定位、语言风格、目标客户和核心价值，确保所有内容输出保持一致的品牌调性。

### steering/beauty-salon-operations.md
Agent 的核心指导文档，定义了 Agent 的角色、职责和工作原则。此文档会自动加载到每次对话中。

### steering/content-templates.md
包含各类内容模板：
- 小红书文案模板
- 抖音/视频号脚本模板
- 微信公众号文章模板
- 朋友圈文案模板
- 活动文案模板

使用时在对话中提及 `#content-templates.md` 即可加载。

### steering/marketing-calendar.md
完整的年度营销日历，包括：
- 12个月的重点营销节点
- 每月固定营销动作
- 每周内容发布计划
- 特殊节点营销策划

使用时在对话中提及 `#marketing-calendar.md` 即可加载。

### steering/customer-service-guide.md
客户服务完整指南，包括：
- 常见咨询应对话术
- 客户异议处理策略
- 客户分类与维护方法
- 投诉处理流程
- 客户关怀时间节点

使用时在对话中提及 `#customer-service-guide.md` 即可加载。

## 💡 使用技巧

1. **明确需求**：告诉 Agent 你需要什么类型的内容、针对什么平台、什么主题
2. **提供背景**：说明你的美容院特色、目标客户、当前活动等信息
3. **引用模板**：使用 `#文件名` 来引用相关模板和指南
4. **迭代优化**：根据 Agent 的输出进行调整和优化

## 🔧 自定义配置

你可以根据自己美容院的实际情况修改以下内容：

1. **brand_tone.md**：调整品牌定位和目标客户群体
2. **content-templates.md**：添加你自己的内容模板
3. **marketing-calendar.md**：根据本地特色调整营销节点
4. **customer-service-guide.md**：补充你的特色服务话术

## 📝 示例对话

**用户**：帮我写一篇小红书文案，推广我们的春季补水护理套餐，价格是原价680元，现在活动价498元，包含3次深层补水护理。

**Agent**：好的！我来为你创作一篇小红书文案...

（Agent 会根据 brand_tone.md 的品牌调性和 content-templates.md 的模板结构，生成符合要求的文案）
