# 选题评分标准

## 评分维度说明

### 1. 时效性 (25%)

**评分标准：**
- 5分：完美契合当前时间节点（节日当天、季节转换期、热点爆发期）
- 4分：较好契合（节日前 3-7 天、季节中期、热点上升期）
- 3分：一般契合（常规时间、通用话题）
- 2分：时效性较弱（过时话题、季节不符）
- 1分：完全不符合当前时间

**示例：**
- 3月8日发"女神节护肤攻略" → 5分
- 3月5日发"女神节预热" → 4分
- 3月发"春季护肤指南" → 4分
- 3月发"夏季防晒" → 2分

### 2. 相关性 (25%)

**评分标准：**
- 5分：直接关联核心产品/服务，能自然引导转化
- 4分：关联主要产品线，转化路径清晰
- 3分：间接相关，需要一定引导
- 2分：相关性较弱，转化困难
- 1分：与业务无关

**示例：**
- "我们的补水项目如何改善干燥肌" → 5分
- "春季补水的重要性" + 项目推荐 → 4分
- "如何判断自己是干性肌肤" → 3分
- "明星的护肤秘诀" → 2分

### 3. 吸引力 (20%)

**评分标准：**
- 5分：标题和内容极具吸引力，能引发强烈好奇或共鸣
- 4分：标题吸引人，内容有价值
- 3分：中规中矩，有一定吸引力
- 2分：平淡无奇，难以引起注意
- 1分：标题和内容都缺乏吸引力

**吸引力要素：**
- 痛点共鸣："你是不是也有这些护肤困扰？"
- 好奇心："为什么你的护肤品没效果？"
- 对比冲击："28岁 vs 38岁，差别竟然这么大"
- 实用价值："5个护肤技巧，立刻见效"
- 真实案例："她用3个月改善了10年的痘印"

### 4. 可执行性 (15%)

**评分标准：**
- 5分：素材现成，30分钟内可完成
- 4分：需要简单准备，1-2小时可完成
- 3分：需要一定准备，半天可完成
- 2分：需要较多准备，需要1天
- 1分：执行难度大，需要多天准备

**考虑因素：**
- 是否有现成素材（照片、视频、案例）
- 是否需要客户配合
- 是否需要专业拍摄
- 文案创作难度
- 后期制作复杂度

### 5. 转化潜力 (15%)

**评分标准：**
- 5分：能直接促成预约/到店，转化路径极短
- 4分：能引导咨询，转化路径清晰
- 3分：能提升品牌认知，间接促进转化
- 2分：主要是曝光，转化较弱
- 1分：纯内容，无转化设计

**转化设计：**
- 限时优惠："本周预约享8折"
- 新客福利："首次体验仅需198元"
- 稀缺性："仅剩5个名额"
- 行动号召："评论区留言，免费获取护肤方案"
- 降低门槛："免费皮肤检测"

---

## 综合评分计算

**公式：**
```
总分 = 时效性×0.25 + 相关性×0.25 + 吸引力×0.20 + 可执行性×0.15 + 转化潜力×0.15
```

**评级标准：**
- 90-100分：优秀选题，强烈推荐，优先级最高
- 80-89分：良好选题，推荐执行，优先级高
- 70-79分：合格选题，可以执行，优先级中
- 60-69分：一般选题，根据情况选择，优先级低
- 60分以下：不推荐

## 评分示例

### 示例 1: 节日营销选题

**选题：** "三八女神节｜宠爱自己，从一次专业护理开始"

| 维度 | 得分 | 理由 |
|------|------|------|
| 时效性 | 5 | 节日当天发布，完美契合 |
| 相关性 | 5 | 直接推广护理项目 |
| 吸引力 | 4 | 节日氛围+情感共鸣 |
| 可执行性 | 5 | 提前准备好素材，快速发布 |
| 转化潜力 | 5 | 节日优惠，转化路径短 |

**总分：** 4.85 → 97分 → 优秀选题 ⭐⭐⭐

---

## 不同场景的评分侧重

### 新客获取期
- 提高"吸引力"和"转化潜力"权重
- 优先选择体验活动、优惠福利类选题
- 降低门槛，快速转化

### 品牌建设期
- 提高"相关性"和"吸引力"权重
- 优先选择教育型、专业展示类选题
- 建立信任，长期价值

### 活动推广期
- 提高"时效性"和"转化潜力"权重
- 优先选择活动相关、限时优惠类选题
- 集中火力，快速转化

### 日常运营期
- 平衡各维度权重
- 多样化选题类型
- 保持稳定输出

---

## 使用建议

1. **每日选题时**：优先推荐 80 分以上的选题
2. **周计划时**：确保每周有 2-3 个高分选题
3. **月计划时**：平衡不同类型，避免单一化
4. **效果复盘时**：对比评分与实际表现，优化评分标准

## 快速评分工具

使用以下问题快速评估选题：

1. ✅ 现在发这个内容合适吗？（时效性）
2. ✅ 能自然引导到我们的项目吗？（相关性）
3. ✅ 目标客户会感兴趣吗？（吸引力）
4. ✅ 我能在短时间内完成吗？（可执行性）
5. ✅ 能促成预约或咨询吗？（转化潜力）

**如果 4-5 个问题回答"是"** → 优秀选题，立即执行
**如果 3 个问题回答"是"** → 良好选题，推荐执行
**如果 2 个问题回答"是"** → 一般选题，谨慎选择
**如果 1 个或更少** → 不推荐
