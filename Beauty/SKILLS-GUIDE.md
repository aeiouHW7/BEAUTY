# 营销技能使用指南

## 已安装的营销技能

我已经为你的美容院项目安装了 8 个专业营销技能，这些技能会自动读取 `.agents/product-marketing-context.md` 中的产品信息，为你提供更精准的营销建议。

### 📝 内容创作类

#### 1. copywriting（文案写作）
**用途**: 撰写各类营销文案
**适用场景**:
- 小红书种草文案
- 朋友圈推广文案
- 活动宣传文案
- 产品介绍文案

**使用示例**:
```
帮我写一篇小红书文案，推广我们的春季补水护理套餐
```

#### 2. social-content（社交媒体内容）
**用途**: 创建和优化社交媒体内容
**适用场景**:
- 小红书内容策划
- 抖音视频脚本
- 微博/朋友圈内容
- 社交媒体日历

**使用示例**:
```
帮我制定本周的小红书和抖音内容发布计划
```

#### 3. content-strategy（内容策略）
**用途**: 规划整体内容营销策略
**适用场景**:
- 月度/季度内容规划
- 内容主题选择
- 内容分发策略
- 内容效果评估

**使用示例**:
```
帮我制定下个月的内容营销策略，重点推广抗衰老项目
```

#### 4. email-sequence（邮件序列）
**用途**: 创建自动化邮件/微信消息序列
**适用场景**:
- 新客欢迎序列
- 会员培育流程
- 沉睡客户唤醒
- 节日问候自动化

**使用示例**:
```
帮我设计一个新客户的5条微信消息欢迎序列
```

### 🎯 转化优化类

#### 5. page-cro（页面转化优化）
**用途**: 优化营销页面的转化率
**适用场景**:
- 小程序首页优化
- 活动落地页优化
- 预约流程优化
- 公众号文章转化

**使用示例**:
```
帮我优化小程序的首页，提高预约转化率
```

#### 6. lead-magnets（引流磁铁）
**用途**: 创建吸引客户的免费资源
**适用场景**:
- 免费皮肤检测
- 护肤知识手册
- 美容小工具
- 限时体验活动

**使用示例**:
```
帮我设计一个"春季护肤指南"电子书作为引流磁铁
```

### 🧠 策略研究类

#### 7. customer-research（客户研究）
**用途**: 分析和理解目标客户
**适用场景**:
- 客户访谈分析
- 评价反馈整理
- 客户画像构建
- 痛点需求挖掘

**使用示例**:
```
帮我分析这些客户评价，找出他们最关心的问题
```

#### 8. marketing-psychology（营销心理学）
**用途**: 应用心理学原理优化营销
**适用场景**:
- 定价策略设计
- 文案心理触发
- 客户决策优化
- 活动机制设计

**使用示例**:
```
用心理学原理帮我设计一个转介绍奖励机制
```

## 如何使用这些技能

### 方法 1: 直接提问（推荐）
直接向我提出你的营销需求，我会自动调用相应的技能：

```
"帮我写一篇小红书文案推广补水项目"
→ 自动使用 copywriting + social-content 技能

"优化我们的预约流程，提高转化率"
→ 自动使用 page-cro 技能

"分析客户为什么不复购"
→ 自动使用 customer-research 技能
```

### 方法 2: 引用上下文
在提问时引用产品营销上下文，获得更精准的建议：

```
基于 #product-marketing-context.md，帮我策划三八妇女节活动
```

### 方法 3: 组合使用
复杂任务可以组合多个技能：

```
帮我完成以下任务：
1. 分析目标客户的核心痛点（customer-research）
2. 基于痛点写一篇小红书文案（copywriting）
3. 设计一个引流活动（lead-magnets）
4. 优化活动落地页（page-cro）
```

## 实战案例

### 案例 1: 策划新客体验活动

**需求**: 设计一个吸引新客户的体验活动

**提问**:
```
基于 #product-marketing-context.md，帮我策划一个新客体验活动：
- 目标：本月新增50个新客户
- 预算：3000元
- 可用项目：补水护理、皮肤检测
- 需要：活动方案 + 小红书文案 + 朋友圈文案
```

**涉及技能**: content-strategy, copywriting, social-content, lead-magnets

### 案例 2: 提升客户复购率

**需求**: 降低客户流失，提高复购

**提问**:
```
我们的客户复购率只有40%，帮我：
1. 分析可能的原因
2. 设计一个客户回访流程
3. 创建一个会员培育的微信消息序列
4. 写一个唤醒沉睡客户的文案
```

**涉及技能**: customer-research, email-sequence, copywriting, marketing-psychology

### 案例 3: 打造爆款内容

**需求**: 创作高互动的社交媒体内容

**提问**:
```
帮我创作一篇能在小红书爆火的内容：
- 主题：夏季防晒护理
- 目标：获得1000+点赞，100+收藏
- 要求：有价值、有共鸣、有转化
```

**涉及技能**: social-content, copywriting, marketing-psychology, content-strategy

## 技能协同工作原理

所有技能都会首先读取 `.agents/product-marketing-context.md`，了解你的：
- 业务模式和服务项目
- 目标客户特征
- 品牌定位和调性
- 营销目标和策略
- 竞争优势

然后基于这些信息提供定制化的营销建议，确保所有输出都符合你的品牌调性和业务目标。

## 更新产品上下文

随着业务发展，你可以随时更新 `.agents/product-marketing-context.md`：

```
帮我更新产品营销上下文：
- 新增了"光电美肤"高端项目线
- 目标客户扩展到18-25岁学生群体
- 新开通了视频号渠道
```

## 技能文件位置

所有技能文件都存储在 `.agents/skills/` 目录下：

```
.agents/
├── product-marketing-context.md    # 产品营销上下文（核心）
└── skills/
    ├── copywriting/                # 文案写作
    ├── social-content/             # 社交媒体
    ├── content-strategy/           # 内容策略
    ├── email-sequence/             # 消息序列
    ├── page-cro/                   # 转化优化
    ├── lead-magnets/               # 引流磁铁
    ├── customer-research/          # 客户研究
    └── marketing-psychology/       # 营销心理学
```

## 常见问题

**Q: 技能会自动激活吗？**
A: 是的，当你的问题匹配某个技能的使用场景时，技能会自动激活。

**Q: 可以同时使用多个技能吗？**
A: 可以，复杂任务会自动组合多个技能协同工作。

**Q: 如何知道使用了哪个技能？**
A: 我会在回复中说明使用了哪些技能，以及为什么使用它们。

**Q: 技能的输出可以修改吗？**
A: 当然，你可以要求我调整任何输出，或者基于特定技能重新生成。
